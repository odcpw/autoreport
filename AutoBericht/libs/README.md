# Bundled Libraries

Place offline copies of all third-party libraries required by AutoBericht here. Suggested structure:

- `markdown-it/markdown-it.min.js`
- `codemirror/codemirror.js` (and CSS, language packs)
- `ajv/ajv7.bundle.js`
- `pagedjs/paged.polyfill.js`
- `pptxgenjs/pptxgen.min.js`
- `pica/pica.min.js`
- `hotkeys/hotkeys.min.js`
- `papaparse/papaparse.min.js`

Each subfolder should include a `LICENSE` and `README` extracted from the original distribution to satisfy offline compliance.
