# AutoBericht Legacy UI

This folder contains the legacy three-tab MVP UI (Import / AutoBericht / Export).
It is kept for reference and fallback, but the current direction is the minimal
editor in `AutoBericht/mini/`.

## How to run

- Start the local server with `AutoBericht/start-autobericht.cmd`
- Open `http://127.0.0.1:<port>/AutoBericht/legacy/`
