ID 1.1.4 — Das betriebliche Geschehen stimmt nicht mit dem Unternehmensleitbild überein

Im Betrieb muss erkennbar sein, ob das Leitbild tatsächlich gelebt wird. GL führt monatlich einen Rundgang mit einer Checkliste durch, in der drei Leitbildpunkte als beobachtbares Verhalten beschrieben sind. Die Checkliste hält Ort, Beobachtung, Abweichung und Sofortmassnahme fest. Die Ergebnisse werden im Protokoll mit Datum und Verantwortlichem erfasst. So sieht das Team Widersprüche sofort. Dadurch werden Abweichungen früher angesprochen und korrigiert.

Abweichungen müssen im Team sauber korrigiert werden, sonst verliert das Leitbild an Wirkung. Linienvorgesetzte führen wöchentlich ein Protokoll, in dem pro Abweichung eine Korrekturvereinbarung festgehalten wird. Das Protokoll enthält Beobachtung, vereinbarte Massnahme, Verantwortlichen und Termin sowie einen Status. Im nächsten Teambriefing wird der Status kurz geprüft und ergänzt. So bleibt die Abweichung nicht liegen. Dadurch wird aus dem Leitbild eine konkrete Verhaltensänderung.

Leitbildkonflikte sollen sich in Führungsentscheidungen zeigen, nicht nur in Gesprächen. GL nimmt quartalsweise einen Leitbildfall in die Führungsbesprechung und entscheidet eine Anpassung einer Arbeitsregel. Das Protokoll enthält Fall, Entscheid, Verantwortlichen, Termin und den Bezug zum Leitbildpunkt. Linienvorgesetzte kommunizieren die Regel im Team und lassen die Kenntnisnahme auf einer Checkliste bestätigen. Das macht Entscheidungen nachvollziehbar. Somit zeigt sich, dass das Leitbild Führungshandeln steuert.

Eine reife Stufe verlangt eine regelmässige Wirksamkeitsprüfung im Betrieb. Der ASA-Spezialist führt halbjährlich eine Prüfung im Betrieb mit einem Formular durch. Das Formular dokumentiert Abweichung, Ursache, Korrektur und Datum der Nachprüfung. Die Ergebnisse werden im Protokoll mit Verantwortlichen und Terminen zusammengefasst. So werden Rückfälle sichtbar. Dadurch lassen sich Massnahmen gezielt nachschärfen.

ID 1.2.4 — Für Arbeitssicherheit besteht keine Budgetposition

Ohne Budget bleibt Sicherheit oft ein Wunsch. GL richtet im jährlichen Budgetprozess eine Budgetposition Arbeitssicherheit ein und gibt sie mit Formular frei. Das Formular enthält Betrag, Zweck, Gültigkeit und Freigabedatum. Linienvorgesetzte können damit Massnahmen planen. Dadurch werden notwendige Massnahmen finanzierbar. Die Priorität wird im Betrieb sichtbar.

Freigaben müssen klar geregelt sein, weil Projekte sonst warten. GL legt jährlich eine Matrix mit Ausgabenlimiten und Entscheidwegen fest. Die Matrix enthält Limit, Massnahmentyp und Gültigkeit. Der ASA-Spezialist prüft die Matrix und dokumentiert Änderungen im Protokoll mit Datum. So weiss jeder, wer entscheiden darf. Dadurch werden Verzögerungen reduziert.

Anträge müssen sauber geführt werden, sonst verliert man den Überblick. Linienvorgesetzte reichen pro Auftrag ein Ausgaben-Formular ein und GL entscheidet. Die Entscheidung wird in einer Pendenzenliste mit Betrag, Massnahme, Status und Termin nachgeführt. Die Liste ist für das Team zugänglich. Mitarbeitende wissen dadurch, was umgesetzt wird. So bleibt keine Massnahme im Tagesgeschäft liegen.

Reife Budgetsteuerung zeigt sich in der Wirkung der Mittel. Der SiBe erstellt jährlich einen Bericht über bewilligte Mittel, umgesetzte Massnahmen und offene Lücken. Der Bericht enthält Beträge, Massnahmen, Gründe für Verzögerungen und Datum. GL prüft den Bericht in der Führungsbesprechung und passt den Budgetrahmen an. Das Ergebnis wird im Protokoll festgehalten. Dadurch wird das Budget näher an den realen Bedarf geführt.

ID 1.5.7 — Keine Sicherheitsstrategie vorhanden

Ohne Strategie fehlt der rote Faden im Sicherheitsjahr. GL erstellt jährlich eine kurze Sicherheitsstrategie auf einem Formular mit drei Prioritäten. Das Formular enthält Priorität, Ziel, Verantwortlichen und Gültigkeit. Die Strategie wird im Betrieb kurz vorgestellt. So wissen alle, worauf es ankommt. Dadurch wird Sicherheit planvoll geführt.

Aus der Strategie müssen konkrete Schritte entstehen. Der ASA-Spezialist überführt die Prioritäten in eine Matrix mit Massnahmen, Verantwortlichen und Terminen. GL gibt die Matrix jährlich frei und dokumentiert Änderungen im Protokoll mit Versionsdatum. Linienvorgesetzte nutzen die Matrix für ihre Jahresplanung. So wird klar, wer was bis wann erledigt. Dadurch wird die Strategie in Massnahmen übersetzt.

Fortschritt muss sichtbar bleiben, sonst verliert die Strategie Wirkung. GL bespricht die Strategie quartalsweise in der Führungsbesprechung. Das Protokoll enthält Fortschritt je Priorität, beschlossene Anpassungen und Termine. Linienvorgesetzte kommunizieren die Entscheide im Teambriefing. Die Rückmeldungen der Mitarbeitenden werden kurz protokolliert. So bleibt die Richtung präsent. Somit werden Prioritäten rechtzeitig nachgesteuert.

Eine reife Strategie zeigt sich im Betrieb, nicht nur im Dokument. Der SiBe führt jährlich eine Stichprobe mit Checkliste durch und prüft, ob die Prioritäten sichtbar sind. Die Checkliste enthält Priorität, Beobachtung, Abweichung und Korrektur. GL dokumentiert die Anpassungen auf dem Strategie-Formular mit Datum. Die Ergebnisse werden im Protokoll der Führungsbesprechung zusammengefasst. So entstehen Lernschleifen. Dadurch bleibt die Strategie wirksam.
